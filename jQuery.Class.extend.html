extend

  • function
 

Extends a class with new static and prototype functions. There are a variety of ways to use extend:

Class.extend(fullName, klass, proto)

Extends a class with new static and prototype functions. There are a variety of ways to use extend:

Parameters

  1. fullName {String}Optional

    the classes name (used for classes w/ introspection)

  2. klass {Object}Optional

    the new classes static/class functions

  3. proto {Object}Optional

    the new classes prototype functions

Returns

{jQuery.Class}

returns the new class

// with className, static and prototype functions
$.Class('Task',{ STATIC },{ PROTOTYPE })
// with just classname and prototype functions
$.Class('Task',{ PROTOTYPE })
// with just a className
$.Class('Task')

You no longer have to use .extend. Instead, you can pass those options directly to $.Class (and any inheriting classes):

// with className, static and prototype functions
$.Class('Task',{ STATIC },{ PROTOTYPE })
// with just classname and prototype functions
$.Class('Task',{ PROTOTYPE })
// with just a className
$.Class('Task')
