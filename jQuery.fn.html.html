jQuery.fn.html

  • function
 

Extending the original jQuery().html() to render jQuery.View templates as the content of each matched element.

View.html(content, data)

Extending the original jQuery().html() to render jQuery.View templates as the content of each matched element.

Parameters

  1. content {String | Object | function()}

    A template filename or the id of a view script tag or a DOM element, array of elements, HTML string, or jQuery object.

  2. data {Object}Optional

    The data to render the view with. If rendering a view template this parameter always has to be present (use the empty object initializer {} for no data).

$('#test').html('path/to/template.ejs', { name : 'javascriptmvc' });
