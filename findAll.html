findAll

  • function
 

FindAll attempts to retrieve a list of model(s) from the sets of currently fetched data. If the model(s) were not previously fetched, it will then execute a request on the static 'findAll' method of the model. It returns the deffered object.

function(params, success, error)

FindAll attempts to retrieve a list of model(s) from the sets of currently fetched data. If the model(s) were not previously fetched, it will then execute a request on the static 'findAll' method of the model. It returns the deffered object.

Parameters

  1. params {Object}

  2. success {*}

  3. error {*}

You can listen for 'findAll' to be triggered by binding to the 'findAll' event on the class.
