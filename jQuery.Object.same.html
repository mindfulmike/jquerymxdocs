same

  • function
 

Returns if two objects are the same. It takes an optional compares object that can be used to make comparisons.

Object.same(a, b, compares, aParent, bParent, deep)

Returns if two objects are the same. It takes an optional compares object that can be used to make comparisons.

Parameters

  1. a {Object}

    an object to compare

  2. b {Object}

    an object to compare

  3. compares {Object}Optional

    an object that indicates how to compare specific properties. Typically this is a name / value pair

    $.Object.same({name: "Justin"},{name: "JUSTIN"},{name: "i"})

    There are two compare functions that you can specify with a string:

    • 'i' - ignores case
    • null - ignores this property

  4. aParent {*}

  5. bParent {*}

  6. deep {Object}Optional

    used internally

This function does not work with objects that create circular references.

Examples

$.Object.same({name: "Justin"},
              {name: "JUSTIN"}) //-> false

// ignore the name property
$.Object.same({name: "Brian"},
              {name: "JUSTIN"},
              {name: null})      //-> true

// ignore case
$.Object.same({name: "Justin"},
              {name: "JUSTIN"},
              {name: "i"})      //-> true

// deep rule
$.Object.same({ person : { name: "Justin" } },
              { person : { name: "JUSTIN" } },
              { person : { name: "i"      } }) //-> true
              
// supplied compare function
$.Object.same({age: "Thirty"},
              {age: 30},
              {age: function( a, b ){
                      if( a == "Thirty" ) { 
                        a = 30
                      }
                      if( b == "Thirty" ) {
                        b = 30
                      }
                      return a === b;
                    }})      //-> true
