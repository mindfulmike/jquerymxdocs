String.deparam(params)

Takes a string of name value pairs and returns a Object literal that represents those params.

Parameters

  1. params {String}

    a string like "foo=bar&person[age]=3"

Returns

{Object}

A JavaScript Object that represents the params:

{
  foo: "bar",
  person: {
    age: "3"
  }
}
