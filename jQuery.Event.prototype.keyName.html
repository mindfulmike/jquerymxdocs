jQuery.Event.prototype.keyName

  • function
jquery/event/key
 

Returns a string representation of the key pressed. The following listens to and prevents backspaces being pressed in inputs:

specialevents.keyName()

Returns a string representation of the key pressed. The following listens to and prevents backspaces being pressed in inputs:

Returns

{String}

The string representation of of the key pressed.

$("input").keypress(function(ev){
  if(ev.keyName() == '\b') {
   ev.preventDefault();
  }
});

Keys

The following describes the key values returned by [jQuery.Event.prototype.key].

  • \b - backspace
  • \t - tab
  • \r - enter key
  • shift, ctrl, alt
  • pause-break, caps, escape, num-lock, scroll-loc, print
  • page-up, page-down, end, home, left, up, right, down, insert, delete
  • ' ' - space
  • 0-9 - number key pressed
  • a-z - alpha key pressed
  • num0-9 - number pad key pressed
  • / ; : = , - . / ` [ \ ] ' "
  • f1-12 - function keys pressed

Alternate keys

Use jQuery.event.key to set alternate key mappings for other locales.
