static.validate

  • function
 

Validates each of the specified attributes with the given function. See validation for more on validations.

validations.validate(attrNames, validateProc, options)

Validates each of the specified attributes with the given function. See validation for more on validations.

Parameters

  1. attrNames {Array | String}

    Attribute name(s) to to validate

  2. validateProc {function()}

    Function used to validate each given attribute. Returns nothing if valid and an error message otherwise. Function is called in the instance context and takes the value to validate.

  3. options {Object}

    Options for the validations. Valid options include 'message' and 'testIf'.
