ready

  • function
 

Indicates that all routes have been added and sets $.route.data based upon the routes and the current hash.

route.ready(val, start)

Indicates that all routes have been added and sets $.route.data based upon the routes and the current hash.

Parameters

  1. val {*}

  2. start {Boolean}Optional

By default, ready is fired on jQuery's ready event. Sometimes you might want it to happen sooner or earlier. To do this call

$.route.ready(false); //prevents firing by the ready event
$.route.ready(true); // fire the first route change
