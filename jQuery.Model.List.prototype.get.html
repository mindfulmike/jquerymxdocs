get

  • function
 

Gets a list of elements by ID or element.

list.get(args)

Gets a list of elements by ID or element.

Parameters

  1. args {Object}

    elements or ids to retrieve.

Returns

{$.Model.List}

A sub-Model.List with the elements that were queried.

To fetch by id:

var match = list.get(23);

or to fetch by element:

var match = list.get($('#content')[0])
