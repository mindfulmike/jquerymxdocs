Language Helpers

  • page
jquerymx.lang  

A collection of language helpers for things like String, Objects, etc. JavaScriptMVC has several lightweight language helper plugins.

Object

Methods useful for comparing Objects. For example, if two objects are the same:

$.Object.same({foo: "bar"}, {foo: "bar"});

Observe

Makes an Object's properties observable:

var person = new $.Observe({ name: "Justin" })
person.bind('change', function(){ ... })
person.attr('name', "Brian");

String

String helpers capitalize, underscore, and perform similar manipulations on strings. They can also lookup a value in an object:

$.String.getObject("foo.bar",{foo: {bar: "car"}})

toJSON

Used to create or consume JSON strings.

[jQuery.Vector Vector]

Used for vector math.
