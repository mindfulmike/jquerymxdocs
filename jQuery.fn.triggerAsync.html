jQuery.fn.triggerAsync

  • function
jquery/event/default
 

Triggers an event and calls success when the event has finished propagating through the DOM and preventDefault is not called.

pause.triggerAsync(type, data, success, prevented)

Triggers an event and calls success when the event has finished propagating through the DOM and preventDefault is not called.

Parameters

  1. type {String}

    The type of event

  2. data {Object}

    The data for the event

  3. success {function()}

    a callback function which occurs upon success

  4. prevented {function()}

    a callback function which occurs if preventDefault was called

$('#panel').triggerAsync('show', function() {
   $('#panel').show();
});

You can also provide a callback that gets called if preventDefault was called on the event:

$('panel').triggerAsync('show', function(){
    $('#panel').show();
},function(){ 
    $('#other').addClass('error');
});

triggerAsync is design to work with the Pause-Resume plugin although it is defined in jquery/event/default.
