unbind

  • function
 

Unbinds a listener. This uses jQuery.unbind and works very similar. This means you can use namespaces or unbind all event handlers for a given event:

observe.unbind(eventType, handler)

Unbinds a listener. This uses jQuery.unbind and works very similar. This means you can use namespaces or unbind all event handlers for a given event:

Parameters

  1. eventType {String}

    • the type of event with any optional namespaces. Currently, only change events are supported with bind.

  2. handler {function()}Optional

    • The original handler function passed to bind.

Returns

{jQuery.Observe}

the original observe for chaining.

// unbind a specific event handler
o.unbind('change', handler)

// unbind all change event handlers bound with the
// foo namespace
o.unbind('change.foo')

// unbind all change event handlers
o.unbind('change')
