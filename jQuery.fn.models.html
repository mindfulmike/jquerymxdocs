models

  • function
 

Returns a list of models. If the models are of the same type, and have a [$.Model.List], it will return the models wrapped with the list.

fn.models(type)

Returns a list of models. If the models are of the same type, and have a [$.Model.List], it will return the models wrapped with the list.

Parameters

  1. type {jQuery.Class}Optional

    if present only returns models of the provided type.

Returns

{Array | $.Model.List}

returns an array of model instances that are represented by the contained elements.
